- 使用nrm快速切换npm源
nrm 是一个 NPM 源管理器，允许你快速地在如下 NPM 源间切换：

列表项目
npm
cnpm
strongloop
enropean
australia
nodejitsu
taobao
Install

- sudo npm install -g nrm
- nrm use taobao
